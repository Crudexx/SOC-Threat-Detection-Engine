# SOC Threat Detection Engine 

A Python-based mini SOC project for detecting suspicious authentication activity from system logs. 


## Features
- Detects brute force login attempts 
- Detects successful login after multiple failures 
- Generates CSV alert reports 
- Maps detections to MITRE ATT&CK 


## MITRE ATT&CK Mapping 
- T1110 - Brute Force 
- T1078 - Valid Accounts 


## Run Project 
python analyzer.py 


## Output 
CSV report saved in /output/alerts.csv
